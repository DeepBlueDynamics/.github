### Flagship

**[nemesis8](https://github.com/DeepBlueDynamics/nemesis8)** — Run AI agents in Docker. Four providers (Codex, Gemini, Claude, OpenClaw), 69 MCP tools, HTTP gateway with scheduler, pokeball sealed environments, remote mode. One binary.

```bash
nemesis8 interactive                           # full TUI session
nemesis8 --provider claude interactive         # switch brains
nemesis8 pokeball run hyperagents --prompt "analyze the meta-agent loop"
```
